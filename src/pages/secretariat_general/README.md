# Module Secrétariat Général

Coordination administrative et supervision globale de l'institution.

## Fonctionnalités
- **Structure** : Gestion des facultés et des promotions.
- **Personnel** : Gestion des enseignants et des secrétaires de faculté avec envoi d'email automatique.
- **Résultats** : Centralisation des notes de tous les étudiants.
- **Recours** : Vue d'ensemble sur les contestations de notes.
