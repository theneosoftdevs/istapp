# Module Enseignant

Interface dédiée aux professeurs pour la gestion pédagogique.

## Fonctionnalités
- **Gestion des cours** : Visualisation des cours attribués.
- **Travaux** : Création d'évaluations avec délais et types variés.
- **Saisie des notes** : Carnet de cotes global et synchronisation automatique.
- **Ressources** : Partage de documents avec les étudiants.
