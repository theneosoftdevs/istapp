# Module Étudiant

Ce module permet aux étudiants de gérer leur parcours académique.

## Fonctionnalités
- **Tableau de bord** : Vue d'ensemble des activités récentes.
- **Horaire** : Consultation de l'emploi du temps.
- **Travaux** : Remise de devoirs (PDF, Formulaires, Liens).
- **Notes** : Consultation des résultats et introduction de recours.
- **Ressources** : Téléchargement des supports de cours.
