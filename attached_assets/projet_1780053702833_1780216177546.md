# Fiche Projet - ISTA-Goma

**ISTA-Goma** est une application web universitaire moderne de type **PWA (Progressive Web App)**, conçue selon une approche **mobile-first**. Elle est accessible sur smartphone, tablette et ordinateur, avec la possibilité d'installation directe sur appareil mobile sans passer par les magasins d'applications (Play Store ou App Store).

## 1. Objectif Principal

L'objectif principal de l'application est de digitaliser et centraliser la gestion académique et administrative de l'université. Elle vise à :
* Améliorer l'organisation interne.
* Renforcer la communication entre les différents acteurs universitaires.
* Accroître la rapidité de traitement des opérations académiques.
* Garantir la sécurité et l'intégrité des données.

L'application est une plateforme complète avec des espaces dédiés et des fonctionnalités adaptées pour quatre catégories d'utilisateurs.

---

## 2. Modules et Fonctionnalités par Profil

### Module Enseignants
Espace destiné aux professeurs permettant de :
- Consulter l'horaire de cours.
- Voir les unités d'enseignement attribuées.
- Consulter les listes des étudiants par cours.
- Enregistrer les présences.
- Saisir les notes des interrogations, travaux pratiques et examens.
- Consulter les résultats par promotion.
- Publier des supports pédagogiques.
- Envoyer des annonces ou informations aux étudiants.
- Consulter l'historique des évaluations.
- Recevoir des notifications liées aux activités académiques.
- Accéder rapidement aux informations importantes depuis un mobile.

### Module Secrétariat de Faculté
Espace de gestion administrative académique permettant de :
- Enregistrer les étudiants.
- Gérer les inscriptions académiques.
- Organiser les promotions, filières et options.
- Attribuer les enseignants aux cours.
- Gérer les horaires académiques.
- Encoder et valider les résultats.
- Générer les listes officielles.
- Gérer les dossiers académiques.
- Publier les communications administratives.
- Consulter les statistiques académiques.
- Centraliser les données liées à chaque faculté.
- Assurer la coordination entre les départements.

### Module Étudiant
Portail dédié aux étudiants permettant de :
- Consulter le profil académique.
- Voir les cours attribués.
- Consulter l'horaire.
- Accéder aux résultats.
- Consulter les notes.
- Télécharger les documents académiques.
- Accéder aux supports de cours.
- Recevoir les annonces officielles.
- Consulter les présences.
- Suivre la progression académique.
- Retrouver l'historique universitaire.
- Recevoir les notifications importantes.

### Module Autorités Universitaires (Rectorat)
Espace stratégique pour le pilotage institutionnel permettant de :
- Consulter les statistiques globales.
- Suivre les effectifs étudiants.
- Analyser les résultats académiques.
- Visualiser les performances par faculté.
- Consulter les rapports administratifs.
- Suivre les activités académiques.
- Recevoir les alertes importantes.
- Accéder à des tableaux de bord de suivi.
- Valider certaines opérations (clôtures, délibérations).
- Prendre des décisions à partir de données centralisées.

## 3. Fonctionnalités Générales (Transversales)

L'application intègre les capacités techniques suivantes :
- Authentification sécurisée.
- Gestion stricte des rôles et des permissions.
- Tableau de bord personnalisé selon le profil connecté.
- Notifications en temps réel.
- Recherche rapide.
- Archivage numérique.
- Historique des actions (traçabilité).
- Génération automatique de documents.
- Synchronisation des données.
- Optimisation mobile-first et interface responsive.
- Installation en tant que PWA.
- Accès rapide hors ligne pour certaines données clés (mise en cache).
- Sauvegarde sécurisée.
- Protection rigoureuse des données académiques.

---

## 4. Objectifs Attendus et Impacts

Le système doit permettre d'atteindre les résultats suivants :
- La modernisation globale de la gestion universitaire.
- La centralisation des informations académiques.
- La réduction drastique des procédures manuelles.
- Une meilleure communication entre tous les utilisateurs.
- La réduction de l'utilisation du papier (démarche écologique).
- Un accès rapide et fiable aux informations.
- Une meilleure organisation académique.
- Un meilleur suivi des étudiants tout au long de leur parcours.
- Un pilotage institutionnel efficace et basé sur des données réelles.
- Une expérience utilisateur fluide sur mobile.
- La mise en place d'une plateforme universitaire moderne, fiable et performante.

*Note de conception : L'application privilégie une interface simple, professionnelle, intuitive et rapide, en mettant l'accent sur l'ergonomie mobile et la facilité d'utilisation pour tous les profils.*
